# Cool Agent

🚀 专为个人智能助手打造的开发框架

让开发者快速构建具备记忆、规划和工具使用能力的个性化AI助手。

## 核心特色

### 🤖 智能记忆系统
- 自动管理短期和长期记忆
- 智能上下文压缩，节省token
- 隐私保护，敏感信息自动脱敏

### 🧠 多种思考模式
- 支持 React、思维链等先进范式
- 灵活的任务规划和决策能力

### 🔧 开箱即用的工具集成
- 通过 MCP 协议接入各类工具
- 支持本地服务和远程服务

### 🎯 模型自由切换
- 统一接口适配多种AI模型
- 无需修改代码即可切换模型

## 适用场景

- 个人日程管理助手
- 学习陪伴智能体
- 工作效率提升工具
- 个性化信息处理助手
